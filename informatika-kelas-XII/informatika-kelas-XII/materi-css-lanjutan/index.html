CSS Lanjutan

Selector & Property

Informatika Kelas XII

Tujuan Pembelajaran

  • Memahami selector lanjutan CSS
  • Menggunakan property CSS modern
  • Menerapkan CSS pada website responsif

Review Singkat (Kelas XI)

  • Selector tag, class, dan id
  • Property dasar: color, font-size

Selector Descendant

Memilih elemen di dalam elemen lain

div p {
  color: red;
}

Selector Child ( > )

Hanya memilih anak langsung

ul > li {
  color: blue;
}

Pseudo-class

Digunakan untuk kondisi tertentu

a:hover {
  color: orange;
}
input:focus {
  border: 2px solid green;
}

Property Layout: Display

div {
  display: flex;
}
  • block
  • inline
  • flex

Flexbox Dasar

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Property Box Model

  • margin
  • padding
  • border
div {
  margin: 10px;
  padding: 15px;
}

CSS Responsif

Menyesuaikan tampilan dengan ukuran layar

@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

Latihan Siswa

  1. Buat selector hover pada tombol
  2. Atur layout menggunakan flexbox
  3. Buat tampilan responsif sederhana

Kesimpulan

  • CSS lanjutan membuat website lebih dinamis
  • Selector membantu memilih elemen spesifik
  • Property mengatur tampilan dan layout
