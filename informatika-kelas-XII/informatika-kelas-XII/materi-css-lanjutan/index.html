Pengertian CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan tata letak halaman web yang dibuat dengan HTML.

Fungsi CSS

  • Mengatur warna dan font
  • Mengatur tata letak halaman
  • Membuat tampilan web lebih menarik
  • Memisahkan desain dari struktur HTML

Inline CSS

Inline CSS ditulis langsung di dalam atribut style pada elemen HTML.

<p style="color: red;">Teks merah</p>

Internal CSS

Internal CSS ditulis di dalam tag <style> pada bagian <head>.

<style> p { color: blue; } </style>

External CSS

External CSS ditulis di file terpisah dengan ekstensi .css.

/* style.css */ p { color: green; }

Cara Kerja CSS

  1. Browser membaca file HTML
  2. Browser membaca CSS
  3. CSS diterapkan ke elemen HTML
  4. Halaman web ditampilkan ke pengguna

Kelebihan CSS

  • Website lebih cepat dimuat
  • Desain konsisten
  • Mudah dirawat (maintenance)
  • Hemat waktu pengembangan
  • Kompatibel di berbagai perangkat

Contoh Penerapan CSS

HTML (index.html):

<h1>Judul Website</h1>

CSS (style.css):

h1 { color: blue; }

Penutup

Dengan CSS, tampilan website menjadi lebih menarik, rapi, dan mudah dikembangkan. CSS adalah dasar penting dalam pengembangan web.
